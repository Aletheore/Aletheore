# Governance Documentation

## Purpose

This directory contains informative governance support documents derived from accepted or draft constitutional artifacts.

## Responsibilities

- Preserve readable summaries of governance roles, boundaries, and records.
- Clearly distinguish informative derived documents from authoritative VDPs.
- Link back to the authoritative constitutional source.

## Future Contents

- Role matrices.
- Governance transition summaries.
- Informative governance checklists.
- Future governance templates after they are specified.

## Relationship to Other Directories

Authoritative constitutional specifications live in `constitution/`. Review artifacts live in `docs/reviews/`. This directory may summarize or organize governance information, but it does not override accepted VDPs or governance records.
